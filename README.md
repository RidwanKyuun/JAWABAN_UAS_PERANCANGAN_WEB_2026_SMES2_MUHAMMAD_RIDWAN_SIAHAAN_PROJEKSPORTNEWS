# JAWABAN_UAS_PERANCANGAN_WEB_2026_SMES2_MUHAMMAD_RIDWAN_SIAHAAN_PROJEKSPORTNEWS

SportNews — Fake API Demo
​Proyek SportNews adalah sebuah aplikasi web single-page statis berbasis HTML, Tailwind CSS, dan JavaScript vanilla. Proyek ini mendemonstrasikan bagaimana data yang berasal dari sebuah API (dalam hal ini disimulasikan menggunakan objek JSON statis) dapat dirender, difilter (pencarian), dan disajikan dalam tata letak kartu yang modern dan responsif.
​🚀 Fitur Utama
​Render Data Dinamis: Menampilkan daftar artikel berita olahraga lengkap dengan judul, abstrak, pratinjau isi konten, penulis, kategori, dan tanggal secara otomatis dari sumber data JSON.
​Fitur Pencarian Real-Time: Pengguna dapat mencari berita berdasarkan kecocokan kata kunci pada judul, abstrak, nama penulis, atau kategori tanpa perlu memuat ulang halaman (tanpa refresh).
​Indikator Meta & Paginasi: Menampilkan informasi jumlah artikel yang ditemukan, token data berikutnya (next token), status akhir data (EOF), serta tanggal pembaruan artikel terakhir.
​Desain Responsif & Modern: Dibangun menggunakan Tailwind CSS untuk memastikan tampilan optimal di perangkat seluler (mobile) maupun desktop, lengkap dengan animasi transisi kartu saat di-hover.
​Tautan Langsung Dokumen: Setiap artikel dilengkapi dengan tautan langsung menuju versi cetak HTML maupun dokumen PDF.
​🛠️ Teknologi yang Digunakan
​HTML5: Struktur semantik dasar halaman web.
​Tailwind CSS (via CDN): Framework CSS utilitas untuk desain antarmuka yang cepat dan responsif.
​Font Awesome 6 (via CDN): Koleksi ikon pelengkap elemen visual UI.
​Vanilla JavaScript: Logika manipulasi DOM, pemfilteran array data, pemformatan tanggal lokal (ID), dan interaktivitas menu.
​📂 Struktur Berkas & Data
​Proyek ini bersifat self-contained di dalam satu berkas utama:
​reactjs.html / index.html — Berisi kode struktur HTML, styling, komponen layout, hingga skrip logika JavaScript.
